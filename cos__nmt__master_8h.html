cos_nmt_master.h File Reference

Detailed Description

The CANopen Network Management (NMT) is used to change the state of a CANopen slave node. This module provides the following functionality:

  • NMT state change
  • NMT Error Control (heartbeat)

This file is part of the optional package "Mini-Master".
