void Cos401_DO_Error ( uint8_t  ubEnableV)

Set / Reset Error Condition.

Parameters
ubEnableVset / reset error condition

This function sets / resets the digital output module error condition. Passing a 0 to ubEnableV resets the error condition, passing a 1 to ubEnableV sets the error condition. If the error condition is set, the outputs are set according to the objects 6206h and 6207h.

Index 6202h - Change Polarity.

Parameters
ubSubIndexVsub-index
ubReqCodeVread / write access

This object shall define the polarity of a group of 8 output lines. Output polarity can be inverted individually:

  • 1 = output inverted
  • 0 = output not inverted

If the object is not supported the device behaves accordingly to the default value. The function is called by the protocol stack.

Index 6206h - Error Mode Output.

Parameters
ubSubIndexVsub-index
ubReqCodeVread / write access

This object indicates, whether an output is set to a pre-defined error value (see object 6207h) in case of an internal device failure:

  • 1 = output value shall take the pre-defined condition specified in object 6207h
  • 0 = output value shall be kept if an error occurs

The function is called by the protocol stack.

Index 6207h - Error Value Output.

Parameters
ubSubIndexVsub-index
ubReqCodeVread / write access

On condition that the corresponding Error Mode is active, device failures shall set the outputs to the value configured by this object.

  • 0 = Output is set to 0 in case of fault, if Object 6206h is enabled
  • 1 = Output is set to 1 in case of fault, if Object 6206h is enabled

The function is called by the protocol stack.
