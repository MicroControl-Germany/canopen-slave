cos404ai.h File Reference

Detailed Description

This module implements the objects for analogue inputs (AI) of the device profile CiA 404. An example for the CiA 404 analogue input application interface is given inside the file cos404ai_app.c which is located inside the profile-404 directory.

The following table gives an overview of the objects supported by the CiA 404 analogue input module.

Index Description Function / Variable Application access
6110h AI sensor type Cos404_AI_Type read
6112h AI operating mode Cos404_AI_OpMode read
6126h AI scaling factor Cos404_AI_SetScaleFactor -
6127h AI scaling offset Cos404_AI_SetScaleOffset -
6131h AI physical unit PV Cos404_AI_SetPVDescription write
6132h AI decimal digits PV Cos404_AI_SetPVDescription write
6150h AI status Cos404_AI_SetStatus write
61A0h AI filter type Cos404_AI_FilterType read
61A1h AI filter constant Cos404_AI_FilterOrder read
7100h AI field value 16-bit Cos404_AI_SetFV16 write
7130h AI process value 16-bit Cos404_AI_SetPV16 write
9100h AI field value 32-bit Cos404_AI_SetFV32 write
9130h AI process value 32-bit Cos404_AI_SetPV32 write

ulPhysicalUnitV- Physical value
Returns

The parameter ubChannelV starts at 0 for the first channel, and has the index N-1 for the channel N. The values ulPhysicalUnitV and ubDigitNrV are not checked for any limits.
