void Cos401_AO_Error ( uint8_t  ubEnableV)

Set / Reset Error Condition.

Parameters
ubEnableVset / reset error condition

This function sets / resets the analogue output module error condition. Passing a 0 to ubEnableV resets the error condition, passing a 1 to ubEnableV sets the error condition. If the error condition is set, the outputs are set according to the objects 6443h and 6444h.

Write Analogue Outputs.

Parameters
