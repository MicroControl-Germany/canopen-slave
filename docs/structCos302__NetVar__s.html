Cos302_NetVar_s Struct Reference

Detailed Description

This structure defines the available network variables. Please note that input variables require transmit PDOs (i.e. the symbol COS_PDO_TRM_NUMBER must be greater 0). In the same way output variables require receive PDOs (i.e. the symbol COS_PDO_RCV_NUMBER must be greater 0
