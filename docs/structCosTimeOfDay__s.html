CosTimeOfDay_s Struct Reference

Detailed Description

The CANopen data type TIME_OF_DAY (refer to eCO_DT_TIME_OF_DAY) represents absolute time. It follows from the definition and the encoding rules that TIME_OF_DAY is represented as bit sequence of length 48.

Component ulMilliseconds is the time in milliseconds after midnight. Component uwDays is the number of days since 1 January 1984.
