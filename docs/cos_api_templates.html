CANopen (FD) Slave Templates

The files present inside the application/template-cos directory are used by the CANopen (FD) Slave Code Examples and are a starting point for own application development.

CANopen (FD) Slave template files

cos_conf.h

CANopen (FD) Slave global configuration file

cos_demo_app.h

Demo application functions, used by CANopen (FD) Slave Code Examples

cos_main.c

Template for protocol stack initialisation

cos_mobj.h

Template for manufacturer objects

cos_user.c

Template for protocol stack callback functions
