CANopen FD Slave Protocol Stack

CANopen® is an open layer-7 protocol based on CAN (Controller Area Network). CANopen is mainly applied in machines as embedded network system. Moreover, it is used as a general industrial communication system with a data transfer rate of up to 1 Mbit/s.

The CANopen FD slave protocol stack has been optimized for low resource requirements and is, thus, well suited for intelligent sensors and actuators. The CANopen FD protocol stack offers the complete functionality for integration of the CANopen standards CiA 301, CiA 1301 and CiA 305 into your devices. The following CANopen services are supported:

  • Service Data Object (SDO, Server)
  • Universal Service Data Object (USDO, Server)
  • Process Data Object (PDO)
  • Network Management (NMT)
  • Emergency Message (EMCY)
  • Synchronisation Message (SYNC)
  • Time Message (TIME)
  • Layer Setting Services (LSS, Slave)
