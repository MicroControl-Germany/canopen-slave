Version History for major 3 releases

Version 3.08

  • set error register before EMCY transmission in CAN error case
  • callback function support for segmented SDO transfer
  • fix Rcv-PDO mapping
  • include support for CiA 410
  • add IAP header
  • example for dynamic PDO mapping

Version 3.06

  • include support for CiA 406
  • Bus error passive triggers EMCY only once
  • CAN errors are now tested in the CosTmrEvent() function
  • re-write dynamic PDO support
  • new reload scheme for heartbeat and guarding ticks
  • new API for NVM (EEPROM) access
  • added EMCY inhibit time
  • EMCY messages are sent from queue
  • support for "verify configuration"

Version 3.04

  • values for object 1F80h are also backwards-compatible
  • add support for DS-404 analogue output
  • support Autobaud in LSS module
  • add support for MicroControl manufacturer objects

Version 3.02

  • define behaviour on bus-off condition (function)
  • check values for object 1F80h, changed to V4.0 of CiA 302
  • define behaviour for parameter storage of PDO IDs

Version 3.00

  • bugfix in LSS baudrate setting
  • Two-step init: Reset Application & Reset Communication
  • updated CANpie driver (SJA1000, PIC18Fx8xx family)
  • CAN and timer can be used in poll mode (cos_conf.h)
  • EMCY message on receive buffer overflow
  • Node is initialised after Bootup message (Tx callback)
  • new functions for non-volatile memory introduced
